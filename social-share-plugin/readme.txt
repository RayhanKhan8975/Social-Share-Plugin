== Description ==
The goal of the plugin is to automatically display selected social network(s) sharing buttons in posts and/or on pages.
 * The following Social Networks are available: Facebook, Twitter, Pinterest, LinkedIn, Whatsapp. 

== Installation ==
1. Upload \"social-share-plugins\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Configure the settings in Admin Dashboard
